# AI Agent Config Builder

Internal tool for Mottasl account managers to generate AI Agent configurations.

## Setup

### Environment Variables (add in Vercel)
- `ANTHROPIC_API_KEY` — your Anthropic API key
- `APP_PASSWORD` — the access code your team will use to log in

### Deploy
1. Upload this folder to a GitHub repository
2. Connect the repo to Vercel
3. Add the two environment variables above in Vercel project settings
4. Deploy — done
